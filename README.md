# mcrcon
这是一个使用python的语言中的GUI和rcon网络传输方式来对**我的世界服务器**进行一个远程命令输出(^人^)
# 注意事项
1、目标服务器需把enable-rcon=设置为true<br>
2、rcon.password=这是设置rcon密码的<br>
3、rcon.port=25575这是rcon的端口
# 免责声明
本产品只是一个作为命令输出<br>
rcon这个协议中的传输是明文的**包括密码**
